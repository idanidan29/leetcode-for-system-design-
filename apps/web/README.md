# apps/web

Next.js 15 frontend for SDIP.

## Run locally

```powershell
cd apps/web
npm install
npm run dev
```

Opens on http://localhost:3000.

## Scripts

- `npm run dev` тАФ dev server with hot reload
- `npm run build` тАФ production build
- `npm start` тАФ run the production build
- `npm run typecheck` тАФ `tsc --noEmit`
- `npm run lint` тАФ ESLint
